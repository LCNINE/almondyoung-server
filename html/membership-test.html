1️⃣ 플랜 조회 및 선택

플랜 조회 버튼을 클릭하여 사용 가능한 플랜을 확인하세요.

2️⃣ 현재 구독 상태 조회

구독 상태 조회 버튼을 클릭하여 현재 구독 정보를 확인하세요.

결제 Intent 생성 버튼을 클릭하여 결제 준비를 시작하세요.

MSA 아키텍처: 멤버십 서버가 구독을 생성하고, 필요시 내부적으로 결제 서버를 호출합니다.

🔧 개발자 도구

개발자용: 결제 Intent 상태를 확인할 수 있습니다.

6️⃣ 통합 테스트 (자동화)

🚀 자동 테스트 시나리오

1. 플랜 조회 → 2. 결제 Intent 생성 → 3. 정기결제 실행 → 4. 구독 생성

통합 테스트를 실행하여 전체 플로우를 확인하세요.
