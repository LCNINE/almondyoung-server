# 유저 스코프 가져오기

> 이 가이드는 `<RouteGuard>` 컴포넌트에서 사용되는 `requiredScope`의 타입을 지정하고, 가져오기 위한 가이드입니다.

## 간단 요약

requiredScope에 이 페이지에 접근가능한 스코프들을 적어주세요

```js
<RouteGuard requireRole={['admin', 'master']} requiredScope={['admin:access']}>
  ..
</RouteGuard>
```

## scopes 타입 지정하기

### 1.

[almondyoung-server레포 /libs/roles/src/constants](https://github.com/LCNINE/almondyoung-server/blob/develop/libs/roles/src/constants/index.ts) 파일에있는것을 그대로 복사해서

/src/types/dto/scopes.ts 파일에 복사붙여넣기 해주세요.

### 2.

모든 `page.tsx`에 `RouteGuard` 컴포넌트를 적용해주세요.

> requiredScope={[]} 부분에서 컨트롤 + 스페이스바를 누르면 적용가능한 타입들이 나열됩니다.

예시)

```js
import RouteGuard from '@/components/layout/route-guard';
import RegionalInvoiceTemplate from '@/features/order/regional-invoice/template/RegionalInvoiceTemplate';

// (자체배송) 지역별 출고 페이지
export default function OrderRegionalInvoicePage() {
  return (
    <RouteGuard
      requireRole={['admin', 'master']}
      requiredScope={['admin:access']}
    >
      <RegionalInvoiceTemplate />
    </RouteGuard>
  );
}
```
