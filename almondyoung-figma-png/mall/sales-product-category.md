# Sales Product Category Management (상품 분류 카테고리)

## Overview
This page provides comprehensive category management functionality for products. It features a hierarchical category tree on the left, detailed category settings in the center, and a help panel on the right. Users can create, edit, and organize product categories with extensive configuration options including display settings, templates, SEO, and various visibility controls.

## Features and Interactions

### Category Tree Features
- **Hierarchical structure**: Nested categories with version numbers
- **Checkboxes**: Multi-select capability
- **Edit icons**: Quick edit access for each category
- **Visual selection**: Blue highlight for active category
- **Add new**: Button to create new categories
- **Bulk management**: Partial management status button

### Category Configuration
- **URL display**: Shows the category's web URL
- **Visibility controls**: Multiple display settings for different platforms
- **Platform targeting**: PC & Mobile, Mobile-only options
- **Product sorting**: Various ordering and filtering options
- **Template system**: Flexible template assignment with rich text editor
- **SEO optimization**: Complete meta tag management
- **Preview capability**: See how category appears on different devices

### Template Editor Features
- **Rich text editing**: Image, link, and formatting tools
- **Condition builder**: Add custom conditions
- **Multi-tab interface**: Separate views for different purposes

### SEO Management
- **Toggle control**: Enable/disable SEO settings
- **Visibility options**: Control search engine indexing
- **Meta tags**: Browser title, author, description, keywords
- **Keyword formatting**: Comma or space-separated

## Color Scheme
- **Primary**: Dark navy blue for sidebar
- **Active state**: Bright blue for selected items and buttons
- **Highlight**: Blue background for selected category
- **Action buttons**: Blue for primary actions
- **Info badges**: "NEW" badge in red/orange
- **Required fields**: Orange/red text for mandatory attributes
- **Background**: White/light gray for content panels
- **Edit icons**: Blue arrows for edit actions

## Data Display Patterns
- **Three-column layout**: Tree view, settings panel, help panel
- **Form-based input**: Extensive form fields for configuration
- **Radio button groups**: Mutually exclusive options
- **Checkbox groups**: Multiple selection options
- **Rich text areas**: Template and SEO content editing
- **Tabbed interfaces**: Organize related settings
- **Hierarchical tree**: Visual category structure
- **Tooltips**: Info icons for contextual help
