# Marketing Banner Groups (배너 그룹 정보)

## Overview
This page manages banner groups and their associated banners. It allows users to create banner groups, configure display settings, and add/edit multiple banners within each group with image upload and linking capabilities.

## Features and Interactions

### Banner Management Features
- **Image upload**: Each banner entry has an image thumbnail
- **Drag-and-drop ordering**: Up/down arrows to reorder banners
- **Multiple display options**:
  - Always display
  - Custom date range
- **Link configuration**: URL input for banner click destination
- **HTML code support**: Custom HTML code input for advanced banners
- **Individual banner actions**:
  - Change banner details
  - Replace image
  - Delete banner

### Banner Group Configuration
- Unique code assignment
- Title and description
- Platform-specific sizing (Mobile and PC)

## Color Scheme
- **Primary**: Dark navy blue for sidebar
- **Active state**: Bright blue for selected items
- **Action buttons**:
  - Blue for primary actions (변경)
  - Gray for secondary actions (이미지)
  - Orange for save action
- **Add buttons**: Blue and red circular icons
- **Background**: White/light gray for content areas

## Data Display Patterns
- Form-based input for group settings
- Card-based interface for adding new banners
- Table/list view for existing banners
- Visual thumbnails for quick identification
- Inline editing capabilities
- Clear action hierarchy (edit, image, delete)
