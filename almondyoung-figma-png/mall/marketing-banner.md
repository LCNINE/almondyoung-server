## Features and Interactions

### Banner Group Management
- **Visual thumbnails**: Preview of banner images in each group
- **Image count badge**: Shows number of banners in group (e.g., "3장" = 3 images)
- **Edit functionality**: "수정" (Edit) button for each group
- **Toggle activation**: Green toggle switches to enable/disable banner display
- **Platform-specific notes**: Some banners marked as "PC만 등록된 나타나는 배너입니다" (PC-only display)

### Navigation and Organization
- **Tabbed filtering**: Multiple tabs for different banner categories
- **Visual hierarchy**: Clear grouping of banner information
- **Scrollable list**: Extended list with multiple banner groups

## Color Scheme
- **Primary**: Dark navy blue for sidebar (#1E1B4B or similar)
- **Active tab**: Blue underline for selected tab
- **Active state**: Bright blue for selected menu items
- **Toggle switches**: Green when enabled
- **Action button**: Orange for "Add Banner Group"
- **Badges**: Gray background for image count badges
- **Background**: White/light gray for content areas

## Data Display Patterns
- **Card-based list**: Each banner group as a card with thumbnail(s)
- **Badge indicators**: Visual count of images in group
- **Status controls**: Toggle switches for quick enable/disable
- **Action buttons**: Edit button for each entry
- **Thumbnail previews**: Multiple small images for groups with several banners
- **Metadata display**: Subtitles and descriptions for specific groups
